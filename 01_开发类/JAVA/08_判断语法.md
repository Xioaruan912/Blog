# 判断语句IF

```java
if(true){

}
```

如果对的 就输出 如果错误 直接跳过`if`语句

对于`if` 后面的 括号可以省略

如果只有 一句 那么就可以省略

```
if(关系表达式) 一句语句体
```

如果有多种情况 那么就有下面的格式

```java
if(关系表达式){
	第一个通过
}else if(关系表达式){
	第二个通过
}else{
	全部都不通过 则走这个
}
```

对于`If`内部定义的变量 声明周期 只有在 `{}`内部 如果出了 生命周期结束

# 选择语句switch

```java
switch(week){  //这里的week 是一个得到值的表达式
	case 1:
		做事;
		break; //必须要break 否则 会继续执行下面 case2
	
	case 2:
		做事2;
		break;
	default: //上面内容全部不匹配的时候
		没事情可做;
		break;
}
```

![image-20260119122611227](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260119122611227.png)

1. case后面是真实数据 并且不可以是变量
2. case数据不允许重复
3. break 结束整个 switch语句
4. default是全部不匹配的时候用的
5. 表达式结果可以是 `字符` `整数byte short int` `枚举【后面到面向对象学习】` `字符串` 不可以是 `long` `小数 float double`

## default的省略和位置

1. `case`和 `default` 没有标准的上下级关系 位置可以任意书写
2. `default`可以省略 但是如果都没有匹配 那么就什么输出结果都没有

## Case穿透

1. 如果break没有写 那么就会继续执行 `匹配结果下面所有`的`case`

## switch新特性

1.可以通过箭头标签 `JDK14正式版本`：

```
case 1 -> {
	完整语句
}
```

这种方法 不需要`break` 不会导致`case`穿透

如果只有一句 那么 `{}`括号都不需要

2.如果语句重复 那么可以多个定义：

```
case 1,2,3,4,5 -> {
	完整语句
}
```

3.返回运行的结果：

```java
String res = switch(number){
    case 1,2,3,4,5 -> {
        yield "工作日" //返回的结果是 工作日
    }
    case 6，7 -> {
        yield "休息日"
    }
    default ->{ //通过这个方法 一定需要 default
    	yield "error"
    }
}
//下面 可以得到上面返回的结果
sout(res)
```

# break和continue

break是直接结束循环语句 跳出循环到下一个代码

continue是跳出 `本次` 循环 进入下一次循环
