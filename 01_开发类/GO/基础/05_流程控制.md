# 顺序结构

通过 main函数 顺序自上而下执行 执行完毕main 程序执行结束

# IF

```go
package main

import "fmt"

func main() {
	var str int
	fmt.Scanf("%d", &str)
	if str == 1 {
		fmt.Println("是1")
	} else if str == 2 {
		fmt.Println("是2")
	} else {
		fmt.Println("不是1 或者2")
	}
}

```

## 高级风格

```go
package main

import "fmt"

func main() {
	if str := 2; str == 1 { //高级风格 IF中初始化
		fmt.Println("是1")
	} else if str == 2 {
		fmt.Println("是2")
	} else {
		fmt.Println("不是1 或者2")
	}
}

```

# switch语句

```go
package main

import "fmt"

func main() {
	str := 1
	switch str {  //也可以使用高级语法 switch str := 1;str{}
	case 1:
		fmt.Println("是1")
	case 2:
		fmt.Println("是2")
	default: //处理均不匹配
		fmt.Println("不是1 或者2") 
	}
}

```

`fallthrough` : 连续执行case 也就是穿脱一层 连续输出两个 case

```go
package main

import "fmt"

func main() {
	str := 1
	switch str {
	case 1:
		fmt.Println("是1")
	case 2:
		fmt.Println("是2")
	default:
		fmt.Println("不是1 或者2")
	}
}

```

# goto

任意跳转

```go
package main

import "fmt"

func main() {
	str := 1
AK:
	if str == 0 {
		fmt.Println("str 是0")
	} else {
		fmt.Println("str 是1")
		goto AK
	}

}

```

直接跳转到AK地址
