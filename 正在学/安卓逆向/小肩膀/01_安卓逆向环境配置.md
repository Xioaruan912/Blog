## 线刷包的组成

![image-20260323101701913](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260323101701913.png)

可以理解为我们线刷包有3个部分 `image` `bootloader` `radio`

 `image`  就是我们的安卓系统 如果我们后面要魔改啊什么的都是基于这个进行的

![image-20260323102010417](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260323102010417.png)

`vendor`是手机设备驱动

`system` 就是我们手机镜像

`boot` 就是Linux内核

刷机的前提是我们购买的手机已经进行了 `Bootloader解锁`

刷机教程如下：

```
https://mp.weixin.qq.com/s/1EySfXSucGdiuEBTfLsymA
```

首先就是接触BL锁 这里我们让购买手机的厂商直接解锁即可

1. 选择 【设置 – 系统 – 关于手机 – 版本号】，点击【版本号】7次开启 【开发者选项】

2. 选择 【设置 – 系统 – 开发者选项】，开启【USB调试】

3. 通过adb进行调试 输入`adb reboot bootloader` 会出现一个倒下的机器人

4. `fastboot devices` 验证即可 如果没有输出 那么需要安装google驱动

   ![image-20260405151103286](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405151103286.png)

设备管理器中 可以发现有告警 需要去AS中下载

![image-20260405151558317](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405151558317.png)

这样就可以了

5. 下面输入`fastboot devices`验证

![image-20260405151654672](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405151654672.png)

6. 开始刷机 只需要点击我们刷机包里面的bat文件即可

![image-20260405151810614](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405151810614.png)

正常来说等待2-3分钟刷机结束

![image-20260323103043356](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260323103043356.png)

只要我们的 bootloader刷成功 那么就不会出错 如果我们连接网络无法成功

```
adb shell settings put global captive_portal_https_url https://www.google.cn/generate_204
```

这是因为 他会默认去访问google 验证

# 真机root

我们刷机完毕后 可以得到一个干净的手机环境 那么我们现在需要解锁最大权限

```
adb shell
su
```

如果提示权限不足 那么就说明没有解锁root

root我们先需要下载`Magisk`

```
https://github.com/topjohnwu/Magisk
```

下载到手机

```
adb install Magisk-v30.7.apk
```

然后我们需要把我们之前的boot.img 发送到手机

![image-20260405152753148](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405152753148.png)

```
 adb push .\boot.img /sdcard/
```

那么我们进入面具开始root

![image-20260405152901423](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405152901423.png)

选择修补一个文件 选择我们刚刚的`img`

![image-20260405152937780](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405152937780.png)

我们现在把修改后的`boot.img`拖出来

```
adb pull /sdcard/Download/magisk_patched-27000_Y0v2C.img
```

那么又是开始刷机 进入我们的bootloader模式

```
adb reboot bootloader
fastboot flash boot magisk_patched-27000_Y0v2C.img
```

这里我是老的机器 所以我使用27 Magisk 才可以刷入

![image-20260405154449181](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405154449181.png)

重启即可 我们重新进入 Magisk 验证

![image-20260405154653529](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260405154653529.png)

这样就是安装成功了

```
adb shell
su
```

如果输出的是`#`就说明是进入`root`

```
marlin:/ $ su
marlin:/ #
```
