# 完全保密性的局限性

这里是说特性存在的局限性

```
 If (Gen, Enc, Dec) is a perfectly secret encryption scheme with message space M and key space K, then |K| ≥ |M|.
```

也就是说 如果要满足 符合完全保密性 那么我们的K空间一定需要大于我们的M空间

# 香农定理



```
Let (Gen, Enc, Dec) be an en-cryption scheme with message space M, for which |M| = |K| = |C|. The scheme is perfectly secret if and only if:
1. Every key k ∈ K is chosen with (equal) probability 1/|K| by Gen.
2. For every m ∈ M and every c ∈ C, there is a unique key k ∈ K such that Enck(m) outputs c.
```

1. Gen必须从所有可能的密钥集合中均匀地选择KEY
2. 所有的m，都使用独一无二的KEY加密输出c
