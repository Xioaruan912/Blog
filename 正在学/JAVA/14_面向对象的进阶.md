# `static`关键字

我们在面向对象的 方法中 不加入 `static` 关键字

如果我们构造 一个 `Student 类` 其中的` teacher`成员变量 是一样的  

那我们需要 初始化的时候 给每个 都赋值一样的` teacher`

`static修饰成员变量`：被这个类的所有对象共享，有一个修改后全部都修改

```java
package src;

public class Student {
    private String name;
    private int age;
    static String teacherName; //所有类 共享一个teacher
}
```

那么我们通过下面方法

```java
package src;

public class Main{
    static void main(String[] args) {
        Student d2 = new Student();
        Student.teacherName = "大老师"; // 为这个类的teacher 设置为"大老师"
        d2.name = "xiaoming";
        d2.age =11;
    }
}
```
