# 切片

slice（切片） 也就是一个变长的序列

底层是数组 返回是引用类型（动态的）

```go
package main

import "fmt"

func main() {
	s1 := []int{1, 2, 3, 4, 5} //创建切片
	fmt.Println(s1)
	s2 := make([]int, 5, 10) // make创建（切片类型，长度，容量）默认是0值
	fmt.Println(s2)
	s3 := make([]int, 5) // make创建（切片类型，长度，容量）长度=容量
	fmt.Println(s2)
}

```

```go
//下面都是slice
arr := [...]int{0,1,2,3,4,5,6}
s := arr[2:6] //取[2,6)
```

slice是引用传递 与数组不同 数组需要传递指针 其实GO不喜欢这种做法

```go
func  updateSlice(arr []int){
	arr[2] =100
}
//这里对传入的内容修改后 会直接反应到 调用函数
```

最底层是一个 数组 所以可以对数组进行切片 可以向后扩展

# 创建slice

```
var s []int //默认0值
s := []int{2,3,4,56}
s := make([]int,16) //告诉长度为10
s := make([]int,16,32) //长度为16 但是预估空间为32
```
