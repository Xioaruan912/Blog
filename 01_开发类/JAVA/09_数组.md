# 数组是什么

对于 `同个数据类`的 多个存储方法 就是数组

java中存在`隐式转化`

`double` 类型的数组 可以存入 `int` `short`等

# 数组的定义

静态初始化：

```
int arr[] = new int{1,2,3}
```

```
int arr[] = {1,2,3}
```

1. 存储连续空间
2. 一旦通过静态定义完毕 长度不允许变化

# 数组的元素访问

数组是以0下标开始的 所以我们通过 下标访问

```
arr[1] // 上面的2
arr[0] // 上面的1
```

# 数组的遍历

我们通过`for` 循环 得到数组的所有内容

```java
for(int i=0;i<=arr.length;i++){

}
```

这里 java中定义出数组后 数组自带一个属性 `length`
