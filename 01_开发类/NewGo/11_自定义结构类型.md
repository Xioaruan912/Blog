# 自定义类型

go语言可以通过 基本数据类型 定义 新的基本类型

```
type Myint int; //设置Myint 为新的数据类型
```

具有和 `int`相同的类型



# 类型别名

语法：

```
type 名字 = 类型
```

我们之前学习的` byte`和` rune`都是 这样定义出来的

```
type byte = uint8
type rune = int32
```

# 区别

虽然只是一个 是否等于 的区别 但是本质上是不同的

1. 自定义类型 是在本包下定义的 如果不导出 别的包无法访问
2. 类型别名 在编译后会消失 会直接转化为他原本的类型
