# 基本数据类型

## 整数类型

| 整数    | 内存占用(字节) |
| ------- | -------------- |
| `byte`  | 1              |
| `short` | 2              |
| `int`   | 4              |
| `long`  | 8              |

如果严格控制内存大小 就需要分析 这个值会不会超过 取值范围

## 小数类型

| 小数     | 内存占用(字节) |
| -------- | -------------- |
| `float`  | 4              |
| `double` | 8              |

## 字符

| 小数   | 内存占用(字节) |
| ------ | -------------- |
| `char` | 2              |

## 小数类型

| 小数      | 内存占用(字节) |
| --------- | -------------- |
| `boolean` | 1              |

![image-20260117095626735](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260117095626735.png)

```java
package src;

public class Main{
    static void main(String[] args) {
        byte a = 1;
        short b = 2;
        int c = 3;
        long d = 4099901923813L; //必须要添加一个 L 结尾
        float e = 1.5F; //必须要添加一个 F 结尾
        double f = 1.2313; //默认类型 所以不需要
        char g = '1'; // 字符
        boolean h = true;
    }
}
```
