map具有下面的性质

1. 内部是无序的，和添加入的顺序是无关的
2. 内部使用` hash表`
3. map是一个引用类型 必须初始化才可以使用 不初始化是 `nil`

# map的定义

格式：

```
var 名字 map[关键字类型][值的类型]
```

我们一般通过make函数对map进行初始化：

```
make(map[关键字类型][值的类型],容量)
```

容量：表示 map要存储多少元素 可以省略 但是一般我们都自己定义好

# map中添加元素

可以初始化后 直接按照键值赋值：

```
var A map[string][int]
A["你好"] = 1
A["再见"] = 0
```

初始化过程中直接赋值：

```
var A map[int][bool]{
	1:true,
	0:false,
}
```

# 判断map关键字是否存在

```
v,ok := A["拜拜"]
```

v ：value 如果不存在就是` nil`

ok ： 是否存在 如果存在就是 `true`

# map的遍历

通过 `range`方法直接遍历即可 但是记得是无序的

# 删除map元素

通过 delete方法：

```
delete(A,"你好")
```
