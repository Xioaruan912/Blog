进程终止是 进程在系统中声明周期的最后一个阶段

```c
int main(){
	return 0;
}
```

其实这样也会 直接返回错误 并且消失

Win32编程中 主要使用 `ExitProcess` 和 `TerminateProcess` 两个都是退出 或者终止 一个是自身 一个是跨进程

# `ExitProcess`

![image-20260218001110185](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260218001110185.png)

```c
#include <Windows.h>


int main() {
    
    //结束自己
	ExitProcess(123456);
	return 0;
}
```

```
(进程 25432)已退出，代码为 123456。
```

可以发现就是返回退出码

我们还可以通过`GetExitCodeProcess` 捕获到 进程退出的退出码 可以通过监控函数监控退出内容

# `TerminateProcess`

![image-20260218001436090](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260218001436090.png)

只要获取句柄 那么就可以通过` TerminateProcess` 退出指定进程

并且句柄必须需要有 `PROCESS_TERMINATE` 访问权限 才允许退出

```c
#include <Windows.h>


int main() {
	STARTUPINFO si = { 0 };
	si.cb = sizeof(si);
	PROCESS_INFORMATION pi = { 0 };
	BOOL OK = CreateProcess(
		TEXT("进程路径"),
		NULL,
		NULL,
		NULL,
		FALSE,
		CREATE_NEW_CONSOLE,
		NULL,
		NULL,
		&si,
		&pi
	);
	if (OK == 0){
		return 0; // 没有运行成功
	}
	system("pause");
	TerminateProcess(pi.hProcess, 200000);
	return 0;
}
```

这两个我们都可以通过`GetExitCodeProcess` 获取退出码 从而对其他内容进行操作
