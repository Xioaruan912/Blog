# 字符串

通过 `byte` 可以定义出一个字符 

但是如何构建字符串呢

```go
package main

import "fmt"

func main() {
	var A string = "ABCASDSAD"
	fmt.Printf("%s", A)

}

```

C中我们知道 是通过数组 实现的 GO 只需要通过关键字 `string` 即可

如果希望通过 C语言那种 通过数组传递

```go
package main

import "fmt"

func main() {
	var A string = "ABCASDSAD"
	var A1 [5]byte = [5]byte{104,101,108,108,111}
	fmt.Printf("%s", A)

}

```

```
ABCASDSAD hello
```

# 循环输出字符串

```go
package main

import "fmt"

func main() {
	var A string = "ABCASDSAD"
	for i := 0; i < len(A); i++ {
		fmt.Printf("%c ", A[i])
	}
}

```

第二种写法

```go
package main

import "fmt"

func main() {
	var A string = "ABCASDSAD"
	for i, ch := range A {
		fmt.Printf("%d %c \n", i, ch)
	}
}

```

可以发现 i自动获取 索引 ch 得到值

如果想输出中文字符

```
只可以通过 range方式实现 如果使用第一个方法 输出的是 byte
```

# 复数类型

实部和虚部 就是GO的新增内容

```go
package main

import "fmt"

func main() {
	var complexValue complex128 = 1.10 + 10i
	fmt.Println(imag(complexValue)) //打印虚部
	fmt.Println(real(complexValue)) //打印实部
}

```

# 数据类型转换

首先我们知道 

```
大数据转小数据 直接切

小数据转大数据 有符号是 符号补 无符号是补0
```

转换方法是

```go
package main

import "fmt"

func main() {
	var i float32 = 1e+3
	int8var := int8(i) //i转换为 int8 然后 赋值
	fmt.Println(int8var)
}

```

结果是 `-24`

转化过程中 数据没有发生任何变化 只是表示范围`扩展`或者`缩小`

# fsprintf转化为字符串

如果希望把内容直接转化为字符串 可以通过fsprintf实现

```go
package main

import "fmt"

func main() {
	var str string
	var i float32 = 1e+3

	str = fmt.Sprintf("%f", i)
	fmt.Println(str)
}

```

`1000.000000`
