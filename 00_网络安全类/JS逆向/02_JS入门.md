## Let

块内作用域 也就是 `{}` 内部的 

```js
function example() {
    var x=1;
    let y=2;
    if(true){
        var x=3;  //这里全局内容
        let y=4;  //这里出去if后 就不是4了
    }
}
```

# 函数

js中的函数定义：

```
function 函数名字() {
	函数体
}

函数名字() //调用
```

可以通过 `var` 赋值

## 自执行函数

```
!function 函数名字() {
	函数体
}()
~function 函数名字() {
	函数体
}()
//只要执行到这里 就自动执行 就不需要 手动执行代码了
```

```
!function run(a) {
	a+=2
	return a
}(function (){
	return 5
}())

//自执行后面的括号内还可以写函数 作为传递的参数
//5 传入 run中作为参数
```



## 传参

```
function func(name) {
	函数体
}

func(1)

```

## 嵌套函数

```
function run(a) {
	a +=1
	funtion b(a){
		a+=2
	}
	return b(a)
}

run(1) //输出 4

```

## 外部调用

```

!function (){

		return 666
	}

}()

console.log(A()) 
//这里执行A就会 执行内部函数 从而输出666
```

# 箭头函数

主要是简单定义出 匿名函数

定义：

```
//()=>{}
(传递的参数) => {
	写代码逻辑
}

```

```
const p = (arg) => {
	console.log(arg*2)
}

p(10) //输出20
```
