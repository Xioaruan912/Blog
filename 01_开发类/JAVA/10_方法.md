# 方法是什么

我们之前 学了好多 功能代码

并且很多重复的代码

所以我们对重复代码 打包 起名字 然后直接调用即可

```
方法是程序执行的最小单元
```

# 方法的格式

```
public static 返回值的类型 方法名字(参数1,参数2){
	方法体
	返回值
}
```

方法就可以看作别的编程语言的函数

```java
package src;


public class Main{
    static void main(String[] args) {
        int sum = Add(1,2);
        System.out.println(sum);
    }
	//方法
    public static int Add( int a, int b){
        return a+b;
    }

}

```

# 方法的重载

如果我们需要实现下面的代码

![image-20260120152555107](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260120152555107.png)

可以发现 代码难度不高 但是 重复性很高 所以我们通过 重载 可以传入 不同参数

所以可以理解为：同一个类中 方法名字相同 参数不同的方法 

![image-20260120152723618](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260120152723618.png)

1. 名字需要相同
2. 参数个数需要不同 【主要通过 参数分析 选择哪个函数】
3. 返回值 不在意
4. 需要在一个类中

```java
public class Main{
    static void main(String[] args) {
        int sum = Add(1,2);
        System.out.println(sum);
    }

    public static int Add( int a, int b){
        return a+b;
    }
    public static int Add( int a, int b,int c){
        return a+b+c;
    }
    public static int Add(int a, int b,int c,int d){
        return a+b+c+d;
    }
    public static int Add(int a, int b,int c,int d,int e){
        return a+b+c+d+e;
    }

}

```
