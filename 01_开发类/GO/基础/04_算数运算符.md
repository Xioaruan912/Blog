# 指针与地址

![image-20251119200327061](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20251119200327061.png)

`&` 可以看作就是取这个变量的地址

`*` 可以看作取这个地址的值

```
var num int = 65 //num变量为 65 
var p *int = &num // &num为地址 p为指向地址的指针
```
