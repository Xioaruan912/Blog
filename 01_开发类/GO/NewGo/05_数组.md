# 声明数组

```
var 数组名字 [大小]类型
```

大小必须是常量 

```
var abce [3]int
```

数组从下标0开始访问 最后一个元素是` len(abce)-1` 如果越界 就会触发 `panic`

# 数组初始化

定义时初始化：

```
var city [2]string{"fuzhou","shanghai"}
```

编译器推导元素个数：

```
var city [...]string{"fuzhou","shanghai"}
```

编译后会自动去数 然后动态构建

使用索引方式初始化：

```
var city [...]string{0:"fuzhou",3:"shanghai",7:"shenzheng"}
```

这里前面是 他的索引下标 通过这个方法会生成 大小为7 的 数组 并且把对应内容写入

# 数组的遍历

`for`遍历

```
var city [...]string{0:"fuzhou",3:"shanghai",7:"shenzheng"}
for i:=0;i<= len(city)-1;i++{
	fmt.Println(city[i])
}
```

`range`遍历

```
var city [...]string{0:"fuzhou",3:"shanghai",7:"shenzheng"}
for i,v := range city {
	fmt.Println(i,v)
}
```

这里的`i`是index 也就是下标 `v`是value 也就是内容 

range会自动返回下标和里面的值 所以其实代码更加简洁
