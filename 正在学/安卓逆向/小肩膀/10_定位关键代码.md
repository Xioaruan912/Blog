# HashMap快速定位

比如我们常用的类型

Vector，HashMap，LinkList 这些数据类型都会经常使用到

那么其实和正常的JSHook一样 直接对系统函数Hook即可

```
Java.perform(function() {
    console.log("脚本已成功注入...");
    var H = Java.use("java.util.HashMap");
    H.put.implementation = function (a,b){
        console.log("用户输入：",a,b);
        return this.put(a,b);

    }
});
```

可以发现 就是找到这个类 Hook `put` 方法 然后再调用函数 Hook就写的很简单了

    console.log("脚本已成功注入...");

    function showStack(){
        console.log("调用堆栈输出：\n",
            Java.use("android.util.Log").getStackTraceString(
                Java.use("java.lang.Throwable").$new()
            )
        )
    }


    var hashmap = Java.use("java.util.HashMap");
    hashmap.put.implementation = function (a,b){
        if(a.equals("username")){
            console.log("用户输入：",a,b);
            showStack();
        }
        return this.put(a,b);

    }
});
```

# ArrayList快速定位

都是开发比较常用的

```

Java.perform(function() {
    console.log("脚本已成功注入...");

    function showStack(){
        console.log("调用堆栈输出：\n",
            Java.use("android.util.Log").getStackTraceString(
                Java.use("java.lang.Throwable").$new()
            )
        )
    }


    var arrayList = Java.use("java.util.ArrayList");
    arrayList.add.implementation = function (a){
        return this.add(a);

    }
});
```

写完就会报错

```
脚本已成功注入...
Error: add(): has more than one overload, use .overload(<signature>) to choose from:
        .overload('java.lang.Object')
        .overload('int', 'java.lang.Object')

```

说明我们需要重载`add`可以选择类型 这里我们两个都写一下

```

Java.perform(function() {
    console.log("脚本已成功注入...");

    function showStack(){
        console.log("调用堆栈输出：\n",
            Java.use("android.util.Log").getStackTraceString(
                Java.use("java.lang.Throwable").$new()
            )
        )
    }
    
    var arrayList = Java.use("java.util.ArrayList");
    arrayList.add.overload('java.lang.Object').implementation = function (a){
        return this.add(a);
    }
    arrayList.add.overload('int', 'java.lang.Object').implementation = function (a,b){
        return this.add(a,b);
    }
});
```

frida -UF -l hello.js -o log.txt
```

# TextUtils

其中的 `isEmpty`方法

如果有调用这个 函数 分析输入是否为空 那么就可以实现分析

```
Java.perform(function() {
    console.log("[*] 脚本已成功注入！时间: " + new Date().toLocaleTimeString());
    var TextUtils = Java.use("android.text.TextUtils");
    TextUtils.isEmpty.implementation = function (a){
        console.log(a);
        return this.isEmpty(a);
    }
});
```

如果没有`return` 那么会报错
