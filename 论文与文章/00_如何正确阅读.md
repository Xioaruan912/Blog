# 如何高效阅读论文

我们得到有关于我们自己研究方向的论文后 我们如何更加高效的阅读呢？

首先我们需要知道论文如何构成

Abstract：这是摘要 一般是概括整个论文包括背景、面临的问题、提出的方案、实验结果及结论

Introduction：这里开始就是正文了，一般介绍为什么要研究这个课题

Related Work：相关工作 回顾前人的研究，定位我自己的工作的位置 既要表达尊重 又要指出不足

Methodology：方法论

Experiments：实验

Conclusion & Future Work：结论与未来

那么这个就是一个论文的基本构成

那么一个简单的说法就是

## 方法0

```
abstract和introduction看清楚背景、问题、挑战，然后看design怎么解决challenges，最后evaluation看看怎么对比的，把握主要的问题关联性
```

## 方法1

下面是读一篇论文前 需要思考的内容

1. 你这个方向的背景是什么
2. 目前有哪些难点等待解决
3. 为了解决这个难点当前都有哪些研究方向（读论文的重点）
4. 这些方法是牺牲了哪些地方来取得进展的
5. 总结出自己打算跟进的方向

虽然在确定自己研究方向后 很难一下想清楚这些内容，但是这里是明显需要知道的 那么基于这个方法后

1. 收集当前研究方向所有最新的论文
2. 先大概读一遍，他们都是怎么解决问题的
3. 挑选自己感兴趣的再精读
4. 这个时候你要读的论文数量不会很多了，必须花时间

这就是得到有关你自己的内容了

# 如何写一篇高水平的学术论文

这里其实就是如何写一个好的论文

- 题目是文章的点睛之笔，一个好的题目是吸引审稿人的关键，所以，写题目的时候要尽可能的将文章的创新之处融合进去。
- 摘要要进行总的概括，先一两句概括文章的背景和做的事情，然后介绍方法、结果和结论，最后点出文章的贡献（偏应用以及政策方面）
- 引言部分需要尽可能地总结现有的经典文献和最新的相关领域研究，一般审稿人会给你提文献梳理深度、新颖程度的问题，在写这部分的时候，首先需要知道你的创新点，然后拿别人研究的不足来反映你的创新，一般按照`大背景-小问题-现有研究如何解决这个问题-有什么局限-因此本研究怎么做`这个总体的思路来写，最后还要列出假设，这部分的假设要与后边的讨论进行联系，在讨论部分说明现有假设合理性。
- 方法部分主要介绍参与者、实验设备、相关的设计方法等，同时说明数据处理以及分析的流程，此处可使用一些流程图之类的 ，主要是讲明白自己的方法。
- 结果部分要注意，只讲结果，不要进一步深入的讨论，深入讨论是后边要做的事情，这部分仅需要展示自己的结果，结合图表，优化自己的可视化方式，来提高文章的美观程度，让审稿人觉得眼前一亮。
- 讨论部分是重点，这部分内容需要按不同的小节来写，将自己的主要发现放到前边，并对比现有研究，与前人研究的相同之处和不同之处之类的，系统的分析为什么会有差距以及相关的贡献和局限性，此处可以引用一些经典的理论来提高深度。
- 结论部分需要对自己全文进行总结，按照条理进行梳理，分条提出自己的研究结论。

上面是一个基本的思路
