# NodeJS环境搭建

JS 逆向基于 NodeJS 所以我们先需要搭建一下 NodeJS 环境

https://nodejs.org/en/download/current

安装后 记得写入 `path` 然后控制台 输入 `node --version`

返回版本号即可

这样环境搭建结束

# 基于 Vscode与快捷键

安装vscode

https://code.visualstudio.com/

安装即可

## Vscode的快捷键

`F5` ：运行

`ctrl + p ` ：快速打开

`ALT + ↑/↓` ：上下移动

`SHIFT + ALT + ↑/↓` ：上下复制

`CTRL + SHIFT + K` ：删除行
