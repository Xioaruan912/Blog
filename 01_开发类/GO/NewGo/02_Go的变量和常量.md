# 变量

程序运行中 可以改变的数值的`名字`

如果我们直接通过地址访问 代码阅读很困难 如果通过有意义的 名字 就可以更快映射 找到



## 声明变量

Go语言中的每一个变量都有自己的类型，并且变量必须经过声明才能开始使用

 同一作用域内不支持重复声明

### 标准声明

格式：

```
var 变量名 变量类型
```

变量以`var` 开头

```go
var name string
var age int
var isOk bool
```

### 批量声明

如果一次性需要声明多个变量 一个一个写很麻烦 我们可以通过批量声明

```go
var (
    a string
    b int
    c bool
    d float32
)

```

## 变量初始化

对变量赋值

声明后 Go 默认对变量赋值 `0`,`nil`,`false`这类

初始化格式：

```
var 变量名 类型 = 表达式
```

单个初始化

```
var name string = "N"
var age int = 18
```

或者一次性初始化多个

```
var name, age = "N", 20
```

#### 类型推导

如果我们不指定 类型 Go编译器自动 推断类型是什么

```
var name = "N" //编译器推断是 string
var age = 18 //编译器推断是 int
```

#### 短变量声明

函数内部我们可以直接通过 `:=` 快速声明

```go
package main

import (
	"fmt"
)
// 全局变量m
var m = 100

func main() {
	n := 10
	m := 200 // 此处声明局部变量m
	fmt.Println(m, n)
}

```

#### 匿名变量

Go声明出来的必须要使用 但是有的时候 函数返回多个 我不需要那么就出问题

Go 使用匿名变量 忽略不适用的 变量

```go
func foo() (int, string) {
	return 10, "N"
}
func main() {
	x, _ := foo() //忽略string 返回
	_, y := foo() //忽略int 返回
	fmt.Println("x=", x)
	fmt.Println("y=", y)
}

```

匿名变量不占用命名空间，不会分配内存，所以匿名变量之间不存在重复声明。 

## 注意

注意事项：

1. 函数外的每个语句都必须以关键字开始（`var`、`const`、`func`等）
2. `:=`不能使用在函数外。
3. `_`多用于占位，表示忽略值。



# 常量

对于我们如果 有值只是表示字符串和常定的值 那么我们通过常量方法定义

常量的声明和变量声明非常类似，只是把`var`换成了`const`，常量在定义的时候必须赋值。

对于`const` 声明的过程中 都需要赋值

```
const pi = 3.1415
const e = 2.7182
```

批量赋值：

```
const (
    pi = 3.1415
    e = 2.7182
)
```

`const`同时声明多个常量时，如果省略了值则表示和上面一行的值相同。 例如：

```
const (
    n1 = 100
    n2
    n3
)
```

# iota

`iota`是go语言的常量计数器，只能在常量的表达式中使用

`iota`在const关键字出现时将被重置为0。

`const`中每新增一行常量声明将使`iota`计数一次

使用iota能简化定义，在定义枚举时很有用。

```go
const (
		n1 = iota //0
		n2        //1
		n3        //2
		n4        //3
	)

```

使用`_`跳过某些值

```
const (
		n1 = iota //0
		n2        //1
		_
		n4        //3
	)

```

`iota`声明中间插队

```
const (
		n1 = iota //0
		n2 = 100  //100
		n3 = iota //2
		n4        //3
	)
	const n5 = iota //0

```

定义数量级 

`<<`表示左移操作

`1<<10`表示将1的二进制表示向左移10位，也就是由`1`变成了`10000000000`，也就是十进制的1024。

同理`2<<2`表示将2的二进制表示向左移2位，也就是由`10`变成了`1000`，也就是十进制的8。

```
const (
		_  = iota
		KB = 1 << (10 * iota)
		MB = 1 << (10 * iota)
		GB = 1 << (10 * iota)
		TB = 1 << (10 * iota)
		PB = 1 << (10 * iota)
	)

```

多个`iota`定义在一行

```
const (
		a, b = iota + 1, iota + 2 //1,2
		c, d                      //2,3
		e, f                      //3,4
	)

```
