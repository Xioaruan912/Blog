# 结构体

引出结构体是可以通过 基本数据类型 表示一个 复杂属性

```
type 结构体名字 struct{
	字段名字 基本数据类型
	字段名字 基本数据类型
	...
}
```

1. 结构体名字 一个包内不允许重复 必须唯一
2. 结构体内 字段名字 必须唯一
3. 基本数据类型 表示 字段的类型

```go
package main

type person struct {
	age  int
	name string
	city string
	// name,city string 可以写到一行内
}

func main() {
	
}

```

# 结构体实例化

也就是 定义出 结构体 只有实例化后 才可以使用内部类型

```
var 名字 结构体名字
```

```go
package main

type person struct {
	age  int8
	name string
	city string
	// name,city string 可以写到一行内
}

func main() {
	var p person
	p.name = "llll"
}

```

### 访问值

```
结构体.属性
```

# 匿名结构体

如果我们 需要在函数内部 快速定义个结构体 使用可以通过 匿名结构体

```
var 名字 struct{Name string;Age int}
名字.Name
名字.Age
```

# 指针类型结构体

通过 `new` 返回的是 后面类型的指针 所以我们直接通过 `new` 可以得到结构体的指针

```go
package main

import "fmt"

type person struct {
	age  int8
	name string
	city string
	// name,city string 可以写到一行内
}

func main() {
	p := new(person)
	fmt.Printf("%#v", p)
}

```

输出：

```
&main.person{age:0, name:"", city:""}
```

可以发现 就是返回的是一个指针 由于go 中 指针安全 所以我们直接通过`.` 就可以访问指针内容

# 初始化

1.键值初始化

```go
func main() {
	p := person{
		name:"lll"
		age:18	}
	fmt.Printf("%#v", p)
}

```

2.直接对列表初始化

```
func main() {
	p := person{"lll",18," "}
	fmt.Printf("%#v", p)
}

```
