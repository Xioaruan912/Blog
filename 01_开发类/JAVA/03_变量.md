# 变量

也就是Java中 数据如何存储

`变量`：一个存储会变化的数据的一个容器 是一个空间

![image-20260117093519445](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260117093519445.png)

定义格式：

```
数据类型 变量名字 = 数据值;
  int 	  a	   =   1;
```

![image-20260117093700808](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260117093700808.png)

`数据类型`：约束了 存储空间的大小 如果超过了这个空间 就报错

![image-20260117093826603](https://raw.githubusercontent.com/Xioaruan912/pic/main/image-20260117093826603.png)

```java
package src;

public class Main{
    static void main(String[] args) {

        int wechat_count = 0;
        int alipay_count = 10;
        int card_count = 20;

        int total = wechat_count+alipay_count+card_count;
        System.out.println("总共有 :"+total);
        wechat_count += 10;
        wechat_count -= 2;

        total = wechat_count+alipay_count+card_count;
        System.out.println("新的总共有 :"+total);
    }
}


```

# 注意事项

1. 变量内部只能存储一个值 一个变量 赋值会被覆盖
2. 不允许重复定义
3. 变量使用前 需要赋值
4. 可以被一句 定义了多个变量 多赋值

```
int a,b,c,d; //第一次定义了多个变量
a=b=c=d=10;  //多赋值
```

# 存储规则

任意数据都是按照 二进制数据存储

不同的数据类型 有不同的存储单元

1字节 = 8 bit(位)

| 数据类型 | 存储单元 |
| -------- | -------- |
| `int`    | 4字节    |
