# 通过多态调用成员的特点

```
父类的类型 名字 = new 子类对象()
```

在编译过程中 如果 父类中的变量 存在 那么可以编译成功 并且运行过程中 也是看父类的内容

```java
package src;

public class person {
    public  int age = 00000;
    public void show(){
        System.out.println("person中的show");
    }
    public void Personshow(){
        System.out.println("Personshow");
    }
}

```

```java
package src;

public class student extends person {
    public  int age = 11111;
    public void show(){
        System.out.println("student中的show");
    }
    public void Studentshow(){
        System.out.println("Studentshow");
    }
}
```

```java
package src;

public class Main{
    static void main(String[] args) {
        person per = new student();
        per.Personshow();
    }

}


```
