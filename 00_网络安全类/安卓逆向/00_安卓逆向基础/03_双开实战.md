# 双开

双开也就是 让系统 认为是2个APP 有下面方法

| 原理           |                                                    |
| -------------- | -------------------------------------------------- |
| 修改包名       | 让 系统认为是两个APP 这样就实现了 两个双卡         |
| 修改Framework  | 对于有系统修改权限的手机 可以通过修改实现          |
| 通过虚拟化技术 | 通过虚拟化 让应用程序 在虚拟机运行                 |
| 插件运行       | 利用 反射替换 动态代理 hook系统的一些函数 实现欺骗 |
