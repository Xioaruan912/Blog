# 变量赋值交换

```go
package main

import "fmt"

func main() {
	var (
		a = 100
		b = 10
	)
	fmt.Println(a, b)
	a, b = b, a //交换 ab的值
	fmt.Println("交换后的值=", a, b)
}
```

# 多重赋值

```go
package main

import "fmt"

func main() {
	var a, b, c = 1, 10, 100
	fmt.Println(a, b,c)
	a, b = b, a //交换 ab的值
	fmt.Println("交换后的值=", a, b)
}

```

可以直接通过自动推断 然后输入 多重赋值

# 匿名变量

如果我们声明 一个变量 不使用GO 会报错 但是如果业务就是需要 声明 但是不需要使用

```go
package main

import "fmt"

func main() {
	var a, b, _ = 1, 10, 100 //匿名变量
	fmt.Println(a, b)
	a, b = b, a //交换 ab的值
	fmt.Println("交换后的值=", a, b)
}

```

如果我函数返回3个值 但是我只需要使用两个 就可以通过`_` 作为变量 可以看作时直接遗弃

```
不占用内存 并且 不分配内存
```

# 常量的基础

不允许改变的量就是常量

`const`  用于定义常量

```go
package main

import "fmt"

const tool = 1000

func init() {
	fmt.Println(tool)
}

```

可以发现 这里就是设定常量  以后都不允许被修改

# 常量的定义

```go
const A  = 172 //自动推理定义
const A int = 172 //显式定义
const ( 	//多常量定义
	A = 10000
    B = 200000
)
```

批量定义

```go
package main

import "fmt"

const (
	xiaoxue = 1
	gaozhon
	daxue
)

func main() {
	fmt.Println(xiaoxue, gaozhon, daxue)
}
//输出 ： 1，1，1
```

 只需要 为第一个数据 赋值 就可以传递后面所有的常量

# iota常量生成器

如果我们需要一个批量的自定义数据类型

```go
package main

import "fmt"


const (
	xiaoxue = iota
	gaozhon
	daxue
)

func main() {
	fmt.Println(xiaoxue)
	fmt.Println(gaozhon)
	fmt.Println(daxue)
}

```

`iota`作用 就是 为下面的添加序号

输出为 0 1 2 也就是常量列表的序号 只需要比较序号 就可以的得到结果

# 无类型常量

如果不确定 常量的类型 可以定义为无类型常量

```go
package main

import (
	"fmt"
	"math"
)

const (
	x float32 = math.Pi
	y float64 = math.Pi
)

func main() {
	fmt.Println(x, y)
}

```

# 枚举

```go
package main

import "fmt"


const (
	xiaoxue = iota
	gaozhon
	daxue
)

func main() {
	fmt.Println(xiaoxue)
	fmt.Println(gaozhon)
	fmt.Println(daxue)
}

```

这里就是枚举

可以通过修改 iota的跨度 修改跳跃

```GO
package main

import "fmt"

const (
	xiaoxue = 1 << iota
	gaozhon
	daxue
)

func main() {
	fmt.Println(xiaoxue)
	fmt.Println(gaozhon)
	fmt.Println(daxue)
}

```
