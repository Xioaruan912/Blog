# 数字运算

类型不同 不可以运算 需要转化同类型 才可以计算

## 从小->大

其实就是在前面补0 默认操作 JAVA 底层自动实现

1. 如果有byte / short 类型【无论是不是同类型】 自动转化 `int`
2. 再把取值范围提升

## 从大->小

这里是强制转化 JAVA不会自动生成

并且强制转化 会对 数据精确值 变差

```
目标数据类型 变量名 = (目标数据类型)被转化的数据
```

例子：

```java
int a = 10;
byte b = (byte)a;
```

# 比较运算符

| 符号 | 内容                            |
| ---- | ------------------------------- |
| ==   | 是不是相等 返回 boolean         |
| !=   | 是不是不相等 返回 boolean       |
| >    | 是不是大于 返回 boolean         |
| <    | 是不是小于 返回 boolean         |
| >=   | 是不是大于或者等于 返回 boolean |
| <=   | 是不是小于或者等于 返回 boolean |

# 逻辑运算符

`&` ： 与 也就是 而且 符号左右 同时true 则true

`|` ： 或 或者 符号左右只要有一个 成立 那么就 true

`!` ： 非 对符号右边内容 取反

# 短路逻辑运算符

`&&` ： 与 也就是 而且 符号左右 同时true 则true 比单个 调高了效率 , 左边 `False` 直接返回`False`

`||` ： 或 或者 符号左右只要有一个 成立 那么就 true 左边true 直接返回true

# 三元运算符

格式如下：

```
关系表达式 ? 成功表达式1 : 失败表达式2
```
