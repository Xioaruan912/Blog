# 定义结构体

只支持封装 不支持 继承和多态 面向接口编程

只有struct	

```go
type treeNode struct{
	data int
	left,right *treeNode
}
```

# 访问结构体修改值

```
var root treeNode
root = treeNode{value:3}
root.left = &treeNode{} //空树
root.right = &treeNode{5,nil,nil} //叶子
root.right.right = new(treeNode) //构建空节点
```

一律通过 点 实现

# 结构定义方法

也就是一个function Go语言所有都是传递值

```go
func (Node treeNode)print(){  //确定接收者是谁 这里传递的是值 
	fmt.pln("%d",node.value)
}
root.print()
```

指针作为方法接收者

```go
func (node *treeNode)updateValue(n){
	node.value = n
}
```

要改变内容 或者 结构过大 都是用指针接收者

如果有指针那么最好都是指针接收者
