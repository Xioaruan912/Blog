# 键盘录入

JAVA 内置 `Scanner` 类 通过 里面的方法 实现`IO`

`import java.util.Scanner;` 通过内置 `util`包 导入 类`Scanner`

```java
package src;

import java.util.Scanner;

public class Main{
    static void main(String[] args) {
        // 通过 Scanner 内置方法 获取 系统输入
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt(); //从键盘输入一个 int
        String d = sc.next(); //从键盘输入一个 字符串
    }
}


```
