# HTTP

超文本传输协议 通过 TCP连接 三次握手四次挥手实现

可以通过 `wireshark`查看 现在一般都是 HTTP 1.1的保本

## 状态代码

`200` ：ok （2xx 一般成功)

`301` ：永久跳转 （3xx 一般都是重定向)

`403` ：永久访问 （4xx 一般都是错误)

`502` ：服务错误 （5xx 一般都是服务器问题/配置问题)

## 报文

#### 请求报文

我发出去给服务器 就是请求保温

```
GET /路径 HTTP版本
Accept: 客户端允许接受的响应类型
Accept-Encoding: 允许接受支持编码
Accept-Language: 允许接受的语言
Cache-Contorl: 控制缓存行为 是否缓存
Connect: 是否持久连接 跳过 TCP 请求过程
Cookie: 保存一个身份
Sec-XXX： UA的一些信息
User-Agent: 用户代理

```

一般传输过程都是如下 POST 会带一个 请求体 

通常通过 JSON格式传输

#### 响应报文

服务器给我的就是响应报文
