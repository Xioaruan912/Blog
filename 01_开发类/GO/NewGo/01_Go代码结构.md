# 赋值

我们对变量赋值 有如下方法

声明后赋值：

```
var a int
a = 1
```

自动赋值：

```
a := 1
```

Go语言中需要声明出来的一定需要使用 如果没有使用 编译错误

# fmt

Go内部内置 GO fmt 格式化方法 我们只需要保存后 自动格式化 即使多个 tab键 

```
go fmt main.go
```

就自动对  go代码 格式化 可以区分 `空格` `tab` 的错乱
